# Investment Preferences & Financial Trends Analysis Dashboard

## Overview
Developed an interactive Tableau dashboard to analyze investment preferences, savings objectives, and financial behavior. The dashboard provides insights into investment patterns, decision-making factors, and financial awareness across different demographic groups.

## Tools Used
- Tableau
- Microsoft Excel

## Dashboard Features
- Total Investors KPI
- Total Investment KPI
- Investment Distribution Analysis
- Savings Purpose Analysis
- Investment Information Source Analysis
- Factors Influencing Investment Decisions
- Interactive Gender and Age Filters

## Key Insights
- Gold recorded the highest investment value among all investment options.
- Wealth Creation emerged as a major savings objective.
- Returns and safety were key factors influencing investment decisions.
- Financial consultants and media sources played an important role in investment awareness.
